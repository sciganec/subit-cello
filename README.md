# subit-cello

## Атлас віолончелі

Тематичний атлас віолончелі, побудований на основі [SUBIT-фреймворку](https://github.com/sciganec/subit-framework).

SUBIT дає каркас `WHO × WHERE × WHEN`, який дозволяє розглядати віолончель не лише як інструмент, а як історію, школу, репертуар, акустичне тіло, оркестрову функцію, культурний знак і поле сучасних трансформацій.

## Структура

```
subit-cello/
├── index.html, styles.css, app.js   — веб-додаток для читання атласу
├── CELLO_SUBIT_MATRIX.md            — повна матриця 64 тематичних станів
├── SUBIT_CELLO_BLUEPRINT.md         — концептуальний каркас релізу
├── GITHUB_RELEASE_DRAFT_UA.md       — чернетка опису релізу
├── chapters_index.md                — зведений перелік глав
└── 11 тематичних глав (*.md)
```

## Глави атласу

| Глава | Тема |
|-------|------|
| `history.md` | Історія віолончелі від ранніх форм до сучасності |
| `organology.md` | Будова, акустика, тілесність гри, струни, смичок, тембр |
| `composers-and-works.md` | Канон авторів і творів від Баха до сучасного корпусу |
| `schools.md` | Національні та педагогічні виконавські школи |
| `orchestra.md` | Функції віолончелі в оркестровому мисленні |
| `chamber-music.md` | Соната, квартет, тріо й мистецтво ансамблю |
| `modern-styles.md` | Кросовер, кіно, електрична віолончель, цифрові медіа |
| `cellists.md` | Видатні виконавці, які змінили історію інструмента |
| `pedagogy.md` | Педагогіка, навчальний канон, передача традиції |
| `diskography.md` | Запис як архів інтерпретацій і пам'ять шкіл |
| `extended-techniques.md` | Розширені техніки і майбутнє звукового словника |

## SUBIT-матриця

64 стани віолончелі описані у `CELLO_SUBIT_MATRIX.md`, згруповані за сезонами:

- **SPRING** (1–16): виникнення, рання історія, становлення
- **SUMMER** (17–32): розквіт, канонізація, великі форми
- **AUTUMN** (33–48): модернізація, злам, експеримент
- **WINTER** (49–64): рефлексія, спадщина, майбутнє

## Веб-додаток

Атлас має вбудований веб-читач з навігацією по главах та інтерактивним переглядом матриці.

Для запуску:

```bash
cd subit-cello
python -m http.server 8000
```

Потім відкрийте `http://localhost:8000`.
