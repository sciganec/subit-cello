## 🌟 Про проєкт

**SUBIT Cello Atlas** — це цифрова екосистема знань про віолончель, яка поєднує глибоке музикознавче дослідження з передовими технологіями візуалізації. Проєкт перетворює абстрактні історичні дані на інтерактивну мандрівку крізь 400 років еволюції інструмента.

> [!IMPORTANT]
> Версія 1.1 повністю позбулася технічного жаргону та зосереджена виключно на музикознавчому, органологічному та виконавському аналізі.

## 🚀 Ключові особливості

- **Interactive 3D Constellation**: Процедурна 3D-модель (React Three Fiber), де кожен вузол — це унікальний стан віолончельної культури.
- **64 Унікальні Статті**: Повний корпус аналітичних текстів, що покривають історію, фізику звуку, видатні школи та сучасний авангард.
- **Матриця Станів**: Гнучка навігаційна сітка з фільтрацією за епохами (Витоки, Канон, Модерн, Рефлексія).
- **Glassmorphic UI**: Сучасний, преміальний дизайн у стилі "Zen Mode" для максимального занурення в матеріал.
- **Deep Content**: Кожен із 64 станів містить розширений опис та глибокий музикознавчий аналіз.

## 🛠 Технологічний стек

- **Core**: React 19 + Vite
- **3D Engine**: Three.js & @react-three/fiber
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Data**: JSON-driven architecture (64 unique analytical entries)
- **Deployment**: GitHub Pages

## 📂 Структура проекту

- `src/components/` — Модульні React-компоненти (CelloViz, Matrix, EntryModal).
- `src/data/` — Головне сховище знань (`matrix_details.json`).
- `src/styles/` — Дизайн-система (Vanilla CSS з використанням CSS Variables).
- `docs/` — Додаткові методичні матеріали та розширені глави.

## 💻 Локальний запуск

```bash
# 1. Клонувати репозиторій
git clone https://github.com/sciganec/subit-cello.git

# 2. Перейти в директорію
cd subit-cello

# 3. Встановити залежності
npm install

# 4. Запустити режим розробки
npm run dev
```
