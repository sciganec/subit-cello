## 🌟 Про проєкт

**SUBIT Cello Atlas** — це не просто база даних, а багатошарове дослідження інструмента крізь призму Агентності (WHO), Простору (WHERE) та Часу (WHEN). Проєкт поєднує глибокий академічний контент із сучасними технологіями 3D-візуалізації.

### Ключові особливості:
- **Cello Constellation (3D)**: Процедурна 3D-модель віолончелі, де кожен вузол є станом у матриці знань.
- **Матриця SUBIT**: 64 аналітичні точки входу в історію та органологію інструмента.
- **11 Тематичних Глав**: Від витоків бароко до сучасних розширених технік (Extended Techniques).
- **Zen Mode**: Режим фокусованого читання для глибокого вивчення матеріалу.
- **Глобальний пошук**: Миттєва навігація по всьому корпусу текстів та даних.

## 📂 Структура репозиторію

- `src/` — Вихідний код (JS, CSS, 3D логіка).
- `content/` — База знань у форматі Markdown.
- `docs/` — Матриця станів та методологічні матеріали.
- `index.html` — Головний інтерфейс.

## 🚀 Як запустити локально

Оскільки атлас динамічно підвантажує дані, вам знадобиться локальний сервер:

1. **Через Node.js** (рекомендовано):
   ```bash
   npm start
   ```
2. **Через Python**:
   ```bash
   python -m http.server 8000
   ```
3. **Через VS Code**: Використовуйте розширення **Live Server**.

## 🛠 Технологічний стек
- **Vanilla JS & HTML5/CSS3** (Glassmorphism design).
- **Three.js** — для 3D-візуалізації сузір'я віолончелі.
- **SUBIT Framework** — методологічна основа аналізу.
