# CELLO SUBIT Matrix

Нижче подано 64 тематичні стани для релізу про віолончель. Кожен стан можна розгорнути в окремий розділ, есе, датасет, нотну добірку або візуальну схему.

## SPRING

1. `ME x EAST x SPRING`: народження віолончелі з родини басових смичкових інструментів, рання будова, корпус, стрій.
2. `ME x SOUTH x SPRING`: ранні способи тримання, посадка, тілесність гри, барокова моторика.
3. `ME x WEST x SPRING`: перші ансамблеві та діалогічні функції віолончелі в basso continuo і камерній музиці.
4. `ME x NORTH x SPRING`: найдавніші свідчення, трактати, іконографія, нотаційні сліди інструмента.
5. `WE x EAST x SPRING`: майстерні Кремони, Брешії та інших центрів, де формується спільна органологічна традиція.
6. `WE x SOUTH x SPRING`: придворні, церковні та міські практики колективного музикування за участю віолончелі.
7. `WE x WEST x SPRING`: ранні ансамблі, де віолончель вчиться взаємодіяти з клавесином, лютнею, скрипкою, голосом.
8. `WE x NORTH x SPRING`: виникнення школи як пам'яті ремесла і передачі практики від майстра до учня.
9. `YOU x EAST x SPRING`: перший діалог інструмента з композитором і майстром, які визначають межі форми і звучання.
10. `YOU x SOUTH x SPRING`: віолончель як співрозмовник співака або соліста в ранніх сценічних умовах.
11. `YOU x WEST x SPRING`: народження камерної взаємності, коли інструмент починає не лише супроводжувати, а відповідати.
12. `YOU x NORTH x SPRING`: раннє осмислення ролі віолончелі в теорії, описах і музичних уявленнях епохи.
13. `THEY x EAST x SPRING`: соціальні та матеріальні умови, що зробили інструмент потрібним у європейській музичній системі.
14. `THEY x SOUTH x SPRING`: інституції раннього музичного життя, де віолончель закріплюється у практиці.
15. `THEY x WEST x SPRING`: розподіл ролей між інструментами в ансамблі до появи повної сольної автономії віолончелі.
16. `THEY x NORTH x SPRING`: історіографічний образ ранньої віолончелі та проблема реконструкції її початків.

## SUMMER

17. `ME x EAST x SUMMER`: зріла будова інструмента, стандарти розміру, струни, підставка, душка, смичок.
18. `ME x SOUTH x SUMMER`: канонічна техніка правої і лівої руки, позиції, штрихи, кантилена, вібрато.
19. `ME x WEST x SUMMER`: повне розкриття виразовості віолончелі в сонаті, концерті, тріо, квартеті.
20. `ME x NORTH x SUMMER`: теоретичне оформлення інструмента як окремої художньої сутності в каноні.
21. `WE x EAST x SUMMER`: формування національних шкіл виготовлення, підбору струн і конструктивних норм.
22. `WE x SOUTH x SUMMER`: консерваторії, класи, ансамблі та оркестри як середовище зрілої виконавської спільноти.
23. `WE x WEST x SUMMER`: камерні та симфонічні колективи, де віолончель стає носієм спільного звукового мислення.
24. `WE x NORTH x SUMMER`: педагогічні лінії, методики і спадкоємність великих шкіл віолончельного мистецтва.
25. `YOU x EAST x SUMMER`: точка зустрічі лютьє, виконавця і композитора в пошуку ідеального балансу форми та звуку.
26. `YOU x SOUTH x SUMMER`: сцена як місце прямого діалогу віолончелі з публікою через співучий тембр.
27. `YOU x WEST x SUMMER`: партнерство віолончелі з фортепіано, скрипкою, квартетом, оркестром і солістом.
28. `YOU x NORTH x SUMMER`: критика, рецензія, музикознавство і слухацьке розпізнавання віолончелі як високого жанру.
29. `THEY x EAST x SUMMER`: економіка майстерень, торгівлі інструментами та культурної інфраструктури канону.
30. `THEY x SOUTH x SUMMER`: місце віолончелі в оркестровому апараті доби класицизму та романтизму.
31. `THEY x WEST x SUMMER`: роль інструмента у великій жанровій системі європейської музики.
32. `THEY x NORTH x SUMMER`: канонізація віолончелі в історії музики, антологіях, навчальних програмах і записах.

## AUTUMN

33. `ME x EAST x AUTUMN`: зміни конструкції, металеві струни, нові матеріали, інструментальні модифікації модерної доби.
34. `ME x SOUTH x AUTUMN`: розширені техніки, флажолети, col legno, sul ponticello, шумові й текстурні ефекти.
35. `ME x WEST x AUTUMN`: входження віолончелі в модернізм, авангард, мінімалізм, кіно і міжжанрові практики.
36. `ME x NORTH x AUTUMN`: нове теоретичне читання віолончелі як лабораторії тембру і тілесного звуку.
37. `WE x EAST x AUTUMN`: професійні спільноти майстрів і виконавців у добу технічної та естетичної перебудови.
38. `WE x SOUTH x AUTUMN`: фестивалі, студії, конкурси, академії як нові простори виробництва віолончельної культури.
39. `WE x WEST x AUTUMN`: колаборації з джазом, роком, електронікою, театром, кіно і медіамистецтвом.
40. `WE x NORTH x AUTUMN`: перегляд шкіл і канону під тиском нових естетик та глобального музичного обміну.
41. `YOU x EAST x AUTUMN`: композитор і виконавець разом штовхають інструмент за межі традиційної органології.
42. `YOU x SOUTH x AUTUMN`: нова сценічна взаємодія з публікою через жест, шум, перформативність і близькість тіла.
43. `YOU x WEST x AUTUMN`: віолончель як партнер у кросоверних діалогах, де жанрові межі навмисно стираються.
44. `YOU x NORTH x AUTUMN`: критична рефлексія над тим, що вважати віолончельним репертуаром і каноном у XX-XXI століттях.
45. `THEY x EAST x AUTUMN`: індустрія, технології запису і виробництва змінюють саму матеріальну долю інструмента.
46. `THEY x SOUTH x AUTUMN`: нові інституції концертного життя переозначують місце віолончелі на сцені.
47. `THEY x WEST x AUTUMN`: масова культура, кіно і цифрові медіа переносять віолончель у ширший публічний обіг.
48. `THEY x NORTH x AUTUMN`: архів модерності, де віолончель фіксується вже не лише в ноті, а й у записі, відео, цифрових базах.

## WINTER

49. `ME x EAST x WINTER`: сучасна віолончель як підсумок століть конструктивної еволюції і поле для майбутніх змін.
50. `ME x SOUTH x WINTER`: повільна робота над звуком, тілом і диханням як зріла практика виконавської рефлексії.
51. `ME x WEST x WINTER`: пізня віолончель між камерністю, медитативністю, саунд-дизайном і постжанровим існуванням.
52. `ME x NORTH x WINTER`: інструмент як об'єкт знання, архіву, реставрації та концептуального переосмислення.
53. `WE x EAST x WINTER`: спільноти майстрів, педагогів і виконавців, що зберігають і переосмислюють ремесло.
54. `WE x SOUTH x WINTER`: освітні практики, майстер-класи, академії і передача досвіду в глобальній мережі.
55. `WE x WEST x WINTER`: міжнародна екосистема віолончелі, де співіснують академія, кросовер, саундтрек і експеримент.
56. `WE x NORTH x WINTER`: колективна пам'ять інструмента в дисках, бібліотеках, цифрових архівах і відкритих репозиторіях.
57. `YOU x EAST x WINTER`: новий діалог між історичним інструментом, сучасним майстром і цифровим дизайном.
58. `YOU x SOUTH x WINTER`: слухач і виконавець зустрічаються в інтимному просторі запису, стриму і персонального прослуховування.
59. `YOU x WEST x WINTER`: віолончель як універсальний співрозмовник у міжкультурному та міжжанровому полі.
60. `YOU x NORTH x WINTER`: питання майбутнього канону, нових антологій і способів опису віолончельної культури.
61. `THEY x EAST x WINTER`: матеріальна спадщина інструмента в умовах глобального ринку, реставрації і музейної логіки.
62. `THEY x SOUTH x WINTER`: сучасний оркестр, сцена і медіа визначають нові режими присутності віолончелі.
63. `THEY x WEST x WINTER`: культурна система використовує віолончель як знак глибини, меланхолії, людяності й інтелекту.
64. `THEY x NORTH x WINTER`: мета-історія віолончелі як феномену, який поєднує інструмент, практику, пам'ять і майбутнє.
